make use of security services like authentication, data integrity and
data confidentiality from a variety of underlying security mechanisms
like Kerberos, using a unified API.

[org.w3c.dom](java.xml/org/w3c/dom/package-summary.md)

Provides the interfaces for the Document Object Model (DOM).

[org.w3c.dom.bootstrap](java.xml/org/w3c/dom/bootstrap/package-summary.md)

Provides a factory for obtaining instances of `DOMImplementation`.

[org.w3c.dom.events](java.xml/org/w3c/dom/events/package-summary.md)

Provides interfaces for DOM Level 2 Events.

[org.w3c.dom.ls](java.xml/org/w3c/dom/ls/package-summary.md)

Provides interfaces for DOM Level 3 Load and Save.

[org.w3c.dom.ranges](java.xml/org/w3c/dom/ranges/package-summary.md)

Provides interfaces for DOM Level 2 Range.

[org.w3c.dom.traversal](java.xml/org/w3c/dom/traversal/package-summary.md)

Provides interfaces for DOM Level 2 Traversal.

[org.w3c.dom.views](java.xml/org/w3c/dom/views/package-summary.md)

Provides interfaces for DOM Level 2 Views.

[org.xml.sax](java.xml/org/xml/sax/package-summary.md)

Provides the interfaces for the Simple API for XML (SAX).

[org.xml.sax.ext](java.xml/org/xml/sax/ext/package-summary.md)

Provides interfaces to SAX2 facilities that
conformant SAX drivers won't necessarily support.

[org.xml.sax.helpers](java.xml/org/xml/sax/helpers/package-summary.md)

Provides helper classes, including
support for bootstrapping SAX-based applications.
