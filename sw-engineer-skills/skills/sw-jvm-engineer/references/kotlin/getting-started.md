# Kotlin Getting Started

## What to know
- Kotlin can be used for console apps, backend services, cross-platform code, Android, and data work.
- The language emphasizes concision without removing explicitness at the boundary.
- Keep the application shape obvious even when the syntax is compact.

## Practical stance
- Use Kotlin idioms where they reduce noise.
- Keep APIs readable for Java callers when interoperability matters.
- Choose the platform-specific runtime model consciously.
