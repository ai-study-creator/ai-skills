# Static Content and Templates

## Static content
- Keep asset directories explicit.
- Use content caching and compression intentionally.
- Separate application routes from static asset routes.

## Templates
- Choose a template engine that matches the rendering model.
- Keep template data small and clearly shaped.
- Avoid pushing business logic into templates.

## Common engines
- HTML DSL
- Freemarker
- Thymeleaf
- Pebble
- Mustache
- Velocity
- JTE
